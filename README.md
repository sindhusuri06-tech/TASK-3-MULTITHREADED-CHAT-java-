# Task 3: Multithreaded Chat Application

## Objective
To build a multithreaded client-server chat
application using Java sockets.

## Description
The server accepts multiple client connections
and handles each client using a separate thread.
Clients send messages to the server in real time.

## Technologies Used
- Java
- Socket Programming
- Multithreading

## Files
- Server.java
- Client.java

## Output
Multiple clients can send messages to the server
simultaneously.

## Conclusion
This project demonstrates real-time communication
and multithreading in Java.
